⚠️ This page won't work as a local file

You're opening this app via file:// (e.g. double-clicked, AirDropped, or opened from Files). Browsers block requests to Gemini and the Panchang API from file:// pages for security — every "Run" button will fail after a long retry, no matter how this file is set up.

Fix it with one of these:

  1. Host this file online (GitHub Pages, Netlify, Vercel, or any web host) and open that web address instead.
  2. Or, on a computer, run a quick local server in the file's folder — e.g. python3 -m http.server 8000 — then open http://localhost:8000/… (or your computer's LAN address from a phone on the same Wi-Fi).
